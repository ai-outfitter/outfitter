# Outfitter Pi extension workspace

This private workspace marks the package boundary for Outfitter's Pi extension code and assets.

The runtime bootstrap extension is still generated by `code/cli/src/cli/commands/PiLoginLaunch.ts` because it embeds launch-specific values (profile paths, trust decisions, model bootstrap state, and generated system prompt paths) and is covered by onboarding tests. Extracting it into a separately built package would be higher risk for this workspace-only change.

Future work can move reusable extension source here once the generated/runtime boundary has a stable API between `@ai-outfitter/outfitter` and this workspace.
